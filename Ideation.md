**교사-학부모 소통 어플**

- **문제상황**: 학부모의 갑질, 부적절한 언행으로 인해 수많은 교사들이 정신적 피해를 겪고 있음
- **아이디어**
    - 교사의 전화번호을 공개하지 않고, 채팅이 가능하게 함
    - AI → 욕설, 비속어, 은어가 담긴 채팅은 전송되지 않음
    - AI → 교사를 향한 명령적인 채팅, 공격적 채팅을 감지, 전송되지 않음
    - AI → 학부모가 명령적, 공격적 채팅을 했을 때 이를 순화시킨 채팅으로 바꿔줌
    - 한달/하루에 정해진 양의 채팅만 보낼 수 있음
    - 공격적 채팅이 n번이상 감지될 시 이용 금지당함, 레드카드
    - 공지, 급식 관련 내용들도 함께 올릴 수 있으면 좋을 것 같음
- **수익모델**
    - 교육청·학교 단위 계약

**고령자를 위한 **AI 약 복용 관리, 병원 일정 관리 어플**

- **문제상황**: 고령자분들이 약을 자주 까먹음(특히 치매환자분들) → AI가 **약 복용 알림 + 음성 안내** + 병원 일정 관리.
- **사용자 흐름**
    1. 보호자가 약 사진 찍어서 앱에 등록
    2. AI → 약 이름/효능 인식 → DB 저장
    3. 매일 시간 맞춰 음성 알림 + 복용 여부 체크
    4. 보호자/자녀는 웹에서 기록 확인 가능
    5. 캘린더로 병원 기록/일정 관리
- **수익 모델**
    - 무료 알림 제공
    - 유료로 "복용 기록 리포트 + 의료 상담 연동"

 **AI 기반 유기동물 매칭 플랫폼**

- **문제 상황**: 보호소에는 많은 유기동물이 있지만, 입양자가 “어떤 아이와 맞을지” 모름.
- **아이디어**:
    - 입양 희망자가 생활 패턴(외출 빈도, 주거 형태, 가족 구성 등)을 입력.
    - AI가 보호소 동물들의 성격/특징과 매칭 → “당신과 잘 맞는 친구” 추천.
    - 해당 동물, 종에 대한 AI 설명 제공
        
        ex. 유기동물이 말티즈 종이다 → 말티즈 종에 대한 지식을 제공
        
- **기술**: 추천 시스템, 설문 기반 프로파일링.
- **수익 모델**: 입양 수수료, 관련 용품/사료 광고 제휴.

    - 기업의 사회적 책임 예산을 여기로 끌어올 수 있을지도
- 다회용기 잃어버리는 것을 줄이기 위해 보증금을 미리 내고, 추후 받아갈 수 있는 식으로 진행

**기능**

- 배민/쿠팡이츠의 UI를 거의 따라가되 조금 더 ‘다회용기’를 이용하는 특성을 살리는 식으로 진행
- 환경 점수
- 환경 기여도 트래킹 : 유저가 다회용기를 몇번 썼는지 확인하고 그만큼 자라는 식물 보여주기

**Operation**

- 수거시스템 최적화 : 비슷한 지역끼리는 한번에 묶어서 수거한다.
- 설거지는 한 곳에서 하면 훨씬 더 위생적으로 관리가 가능할지도

**AI**

- 수거 동선 최적화
- 세척 품질 검수 (근데 이건 컴퓨터 비전이 필요함)

**WHY**

- 대부분의 비공굿은 트위터에서 개인 제작 후 개인 발송의 형태로 이루어짐
    - 사기 / 먹튀 / 품질 저하가 일어나도 계폭하고 다시 계정을 만들면 그만
    - 개인이 관리하므로 여러 관리 미흡이 생김
    - 큰 판에서는 내가 원했던 굿즈가 나온다고 해도 알 수가 없음.
    - 셀러들도 처음부터 제작을 하는 경우에, 매우 막막하고 그에 대한 정보를 얻기가 힘듦
    - 제작 업체 또한 너무나 케바케지만 ( 이미 옛날에 폐업한 곳부터 품질이 별로인 곳, 말이 안 통하는 곳 등등등..) 그에 대한 db가 매우 적음. 보통은 돈받고 쓴 네이버 블로그 정도

**어떤 기능이 필요한가**

- 굿즈 제작 과정 : 수요조사 -> 펀딩 및 선입금 -> 제작 완료 된 후 일괄 배송 가능 시스템
- 셀러들 간 커뮤니티 : 제작업체에 대한 db 쌓기 / 제작 방법에 대한 db
- 장르 간 검색 기능
- 셀러에게 후기를 남길 수 있게
- 본인인증 빡세게

**BM**

- 펀딩 후 1%만 받아도 …. 전체가 상당한 고액이니 괜찮을 것 같음.

**AI**

- 자연어 검색
“괴출 솜깅 게롱”
—> 괴출 : ‘괴담에 떨어져도 출근을 해야 하는구나’ 장르
—> 솜깅 : 솜인형 굿즈
—> 게롱 : ‘게롱’ 이라는 별명을 가진 캐릭터
이런 방식으로, 놓치는 검색이 없게 도와줌.
워낙 은어와 줄임말이 많은 분야이므로,,,,
- ai 제작 가이드 : 대략적인 단가를 짜게끔

**💬 팀원 의견**

- 돈을 먼저 받고 → 서비스와 제작업체 연계
    - if 천만원 → 펀딩받은 돈으로 업체한테 바로 가게끔
- 얼만큼 펀딩이 됐는지 공개, 업체가 제작중인지 다 완성됐는지 서비스가 공개
    
    총대가 공개하는게 아니라 어플리케이션이 공개 like 네이버 펀딩

    → 서비스를 좀 줄이고 메인 기능을 정할 필요가 있다. ex only 비공식굿즈 판매
    
- 굿즈 종류가 많음 → 모두 다 필터링 할지 or 냅둘지 or ai로 검색기능 강화할지를 고민해봐야한다.

## 2. 약 복용 관리 서비스

### 🎯 왜 필요한가?

- 복용해야 할 약이 많고, 복용 시기/방법이 제각각
- 노환, 정신질환, 바쁜 일정으로 제때 복용이 어려움
- 수면제 성분 등으로 장시간 수면 → 복용 누락 발생
- 약 오남용 우려 존재

👉 **약을 제 시간에, 올바른 방법으로 복용하도록 돕는 서비스**

---

### 🔑 주요 기능

1. 약 봉투 촬영
2. **OCR**로 복용 시기/방법/용량 추출
3. AI가 데이터 기반으로 **알람 자동 설정** (사용자 수정 가능)
4. 복용 시간 도래 시 알람 울림
5. 약을 손에 올려 **촬영 인증**

---

### 💰 BM (비즈니스 모델)

- **무료** : 알람 수동 설정, 복용 인증
- **유료** : OCR + AI 기반 알람 자동 설정

---

### 🚀 차별화 포인트

- **복용 패턴 학습**: 자주 까먹는 시간대 → AI가 미리 알람 강화
- **복용 리포트 자동 생성**: 주치의에게 제출 가능
- **보호자/가족 공유**: 복용 여부 원격 확인 가능
